# Portfolio

Portfolio personnel développé avec **Next.js**, **TypeScript** et **Tailwind CSS**.  

## Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- ESLint
- next-sitemap

## Prérequis
- Node.js 18+
- npm (ou pnpm / yarn)

## Installation & exécution

```bash
# Installation des dépendances
npm install

# Lancement en développement
npm run dev
# Ouvrir : http://localhost:3000

# Build production
npm run build

# Lancement en production
npm run start
